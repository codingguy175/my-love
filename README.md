# My Love 🐻💕

A cute interactive love proposal web app built with React + Vite.

## 🚀 Deploy to GitHub Pages

### 1. Install dependencies
```bash
npm install
```

### 2. Update the base URL
In `vite.config.js`, change `/my-love/` to your GitHub repo name:
```js
base: '/your-repo-name/',
```

### 3. Run locally
```bash
npm run dev
```

### 4. Deploy to GitHub Pages

**Option A — automatic with gh-pages:**
```bash
npm run deploy
```

**Option B — manual:**
```bash
npm run build
# Then push the /dist folder to your gh-pages branch
```

### 5. Enable GitHub Pages
Go to your repo → Settings → Pages → Source: **gh-pages branch**
